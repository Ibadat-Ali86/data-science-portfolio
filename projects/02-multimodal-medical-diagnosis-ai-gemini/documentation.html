3. AI Reasoning Strategy

The system uses a structured prompting approach:

  • System Instruction: Defines the AI as a "Senior Medical Consultant" with specific output requirements
  • Low Temperature (0.2): Ensures consistent, accurate medical responses
  • JSON Output Format: Structured response for reliable frontend parsing
  • Confidence Scoring: AI self-evaluates certainty (0-100%)
Why Gemini 2.0 Flash?
It's the only model that natively supports image + audio + text in a single inference call, enabling true cross-modal correlation.

Known Issues & Challenges

1. Sample File Dependency

The "Load Sample Case" feature requires actual files in data/samples/ directory to call the real Gemini API.

Solution: Implemented fallback demo mode with mock results for seamless presentation.

2. Audio Format Compatibility

Gemini 2.0 Flash works best with WAV format. MP3 files may require transcoding.

Solution: Accept both formats but recommend WAV for optimal results.

3. API Rate Limits

Gemini API has usage quotas that may limit throughput in high-volume scenarios.

Solution: Implemented error handling with graceful fallback to demo mode.

4. Large File Uploads

High-resolution X-rays (>10MB) may cause timeout issues.

Solution: Client-side validation and recommended max file sizes in UI.
